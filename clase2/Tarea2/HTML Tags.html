<Tag>

La etiqueta de comentario se utiliza para insertar comentarios en el código fuente, estos no se muestran en los navegadores, solo en consola.

Ejemplo

<!DOCTYPE>

Todos los documentos HTML deben comenzar con la declaración <!DOCTYPE>. Esta no es una etiqueta HTML, sino es una "información" para el navegador sobre que tipo de documento esperar.

Ejemplo

<!DOCTYPE html1>

<a>

Define un hipervínculo que se utiliza para vincular una página a otra. Su argumento más importante es <href> que indica el destino del enlace.

Ejemplo

<a href = "https//www.w3schools.com">

<abbr>

Define una abreviatura o un acrónimo como HTML, CSS, Mr.,Dr.,ASAP,ATM.

Consejo: utilice el atributo global para mostrar la descripción de abreviatura cuando pase el mouse sobre el elemento title.

Ejemplo

<abbr tittle="Three Letter Acronym">TLA</abbr>acrónimo de tres letras

<address>

Define la información de contacto del autor/propietario de un documento o artículo. Puede ser una dirección de correo, URL, dirección física, n° de teléfono, identificador de RRSS, etc. Generalmente se muestra en cursiva.

Ejemplo

<address"> Written by <a href="mailito@example.com>JC</a>

<article>

Especifica contenido independiente y autónomo. Debe tener sentido por sí solo y debe ser posible distribuirlo independientemente del resto del sitio.

    Publicación en el foro
    Entrada en un blog
    Noticia

Nota: el <article> no se muestra como nada especial en el navegador.

Ejemplo

<article> 

              <h2>Google Chrome</h2><p> Google Chrome es un
              buscador desarrollado por Google, liberado en 2008.</p>
              

              </article>
          

        

      

      


      
      

        

          
<aside>

          

            Define un contenido aparte del que se coloca, debe estar
            indirectamente relacionado con el contenido circundante
          

          

            Consejo: <aside> suele colocarse como una
            barra lateral en un documento.
          

          
Ejemplo

          

            <aside>
              

              <h4>Epcot Center</h4> Epcot es un parque temático de
              Walt Disney Resort.</p>
              

              </aside>
          

        

      

      


      
      

        

          
<b>

          
Pone un texto en negrita

          

            Consejo: también puedes utilizar CSS para establecer texto en
            negrita con "font-weight:negrita".
          

          
Ejemplo

          

            
              <p>Este es un texto normal -<b> y este es un texto
              resaltado</b></p>
            
          

        

      

      


      
      

        

          
<base>

          

            Especifica la URL base y/o el destino de todas las URL alternativas
            de un documento. Debe estar dentro del elemento
            <head>
          

          

            Nota: debe tener presente un atributohref o target
            o ambos.
          

          
Ejemplo

          

            <head>
              

              <base href="https://www.w3schools.com" target="_blank">
              

              </head>
          

        

      

      


      
      

        

          
<bdi>

          

            BDI significa aislamiento bidireccional. La etiqueta aísla una parte
            del texto que podría tener formato en una dirección diferente del
            resto del texto externo. Es útil para poner contenido generado por
            el usuario con una dirección de texto desconocida.
          

          
Ejemplo

          

            <ul>
              

              <li>User<bdi>hrefs</bdi>:60 puntos</li>
              

              </ul>
          

        

      

      


      
      

        

          
<bdo>

          

            BDO significa Anulación bidireccional. Se utiliza para anular la
            dirección del texto actual
          

          
Ejemplo

          

            <bdo dir="rlt"> Este texto irá de derecha a
              izquierda.</bdo>
          

        

      

      


      
      

        

          
<blockquote>

          

            Especifica una sección citada de otra fuente. Los navegadores suelen
            aplicar sangría a <blockquote>.
          

          

            Consejo: utilícelo con <q> para citas en
            línea breves.
          

          
Ejemplo

          

            <blockquote
              cite="http://www.worldwildlife.org/who/index.html"> Durate 60
              años, WWF ha trabajado para ayudar a las personas y la naturaleza
              a prosperar. </blockquote>
          

        

      

      


      
      

        

          
<body>

          

            Define el cuerpo del documento. Contiene todo el contenido de un
            documento HTML, como
            encabezados, párrafos, imágenes, hipervínculos, tablas,
              listas,etc.
          

          

            Nota: sólo puede haver un <body> en un
            documento.
          

          
Ejemplo

          

            <body>
              

              <h1>This is heading</h1> <p>This is a
              paragraph</p>
              

              </body>
          

        

      

      


      
      

        

          
<canvas>

          

            Se utiliza para dibujar grágicos sobre la marcha, mediante
            secuencias de comandos.
          

          
Ejemplo

          

            <canvas id="myCanvas">
              

              Tu buscador no soporta el tag de canvas
              

              </canvas>
          

        

      

      


      
      

        

          
encabezados <h1> - <h6>

          

            Las etiquetas <h1> - <h6> se utilizan para definir
            encabezados HTML. 

             <h1>  Define el título más importante.
            

             <h6> Define el encabezado menos importante. 

          


          

            Nota: Utilice solo un <h1> por página;
            éste debe representar el título/asunto principal de toda la
            página.Comience siempre en orden.
          

          
Ejemplo

          

            <h1> encabezado </h1>
