Etiqueta <button>

Definición y uso

La etiqueta <button> realiza un boton en el que puedes hacer clic. Dentro del <button> puedes poner texto y etiquetas como 'i, b, strong, br, img', etc.

Pero no se puede crear con una elemeto <input>.

Tip:

Especifique siempre el atributo type en un <button> para indicar a los navegadores que tipo de botón es.

Ejemplo

<button type="button" >Click Me! </button >

Alguno de sus atributos son:

Attibute Value Description
form form_id Especifica a que formulario pertenece el botón.
name name Especifica un nombre para el botón.
value text Especifica el valor inicial del botón
