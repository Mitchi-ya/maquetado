Etiqueta <menu>

Definición y uso

La etiqueta <menu> define una lista desordenada de contenido. Se uso junto con <li> para crear elementos de menú y junto con <data> que se utiliza para darle una traducción legible por máquina con un contenido determinado.

    Se puede combinar con diferentes etiquetas. Por ejemplo <details, input, label>

    <Details>

    Especifica una lista de opciones predefinidas para un elemento <input>.

    <Input>

    Especifica un campo de entrada donde el usuario puede ingresar datos. Es el elemento más importante.

    Tip:

    Utilice siempte con la etiqueta <label>





    <Optgroup>

    Se utiliza para agrupar opciones relacionadas en un elemento <select>.(lista desplegable) Si tiene una lista larga, las opciones realcionadas son más fáciles de manejar.
