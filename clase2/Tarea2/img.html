Etiqueta <img>'

Definición y uso

La etiqueta <img> se utiliza para colocar una imágen en una página HTML. Las imágenes NO se insertan en una página web; sino que están vinculadas a páginas web. La etiqueta <img> crea un espacio de retenció para la imagen a la que se hace referencia.

Tip:

Especifique siempre el ancho y alto de una imagen, en caso de no hacerlo la página puede parpadear mientras carga.

Sus atributos obligatorios son:

Atribute Description
src Especifica la ruta de la imagen.
alt Especifica un texto alternativo para la imagen, si esta por algún motivo no se puede mostrar.

Ejemplo

<img src="img.jpg" alt ="Descripicón de imagen "

Se puede combinar con diferentes etiquetas.

Por ejemplo <area>'

En la etiqueta <imagen> puede usarse la etiqueta <area>, esta define el área dentro de un mapa de imagen, (donde se puede hacer clic). Los elementos siempre están animados dentro de una etiqueta <map> usando la estructura del ejemplo anterior. Se puede usar con una imagen con varios elementos.
